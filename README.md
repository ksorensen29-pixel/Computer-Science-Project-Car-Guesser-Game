Computer-Science-Project: Car Guesser Game

Group Members: Gael, Iker, Konrad

Overview

This repository contains a car guessing game designed to test and improve players' knowledge of automobiles. The game presents users with an image of a car, and they must correctly identify key details such as the model, manufacturer, and production year.

The project is inspired by popular guessing games like GeoGuessr and Wordle, adapting their core gameplay loop into a car-focused experience.

Introduction

The Car Guesser Game is a web and mobile-based application that challenges players to recognize vehicles from images. The goal is to create an engaging and educational experience for car enthusiasts, with gameplay centered around observation and knowledge recall.

Core Gameplay
The user is shown a random image of a car (often rare or high-end models).
The player must input:
Car model name
Manufacturer
Production year
The game evaluates the answers and provides feedback.

The primary mode is single-player, focusing on individual skill and learning.
